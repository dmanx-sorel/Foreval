                            
                        Для компиляции  Foreval.dll:

  Для Дельфи:
  Скопировать исходники Foreval ".pas" из папки .../Foreval/Foreval в папку .../Foreval/Delphi
      Компилировать Foreval.dpr.

  Для Лазарус:
  Скопировать исходники Foreval ".pas" из папки .../Foreval/Foreval в папку .../Foreval/Lazarus
      Компилировать Foreval.lpr.
  
  Убедитесь, что модуле Foreval_Lib.pas установлен ключ {$DEFINE USEDLL}.
