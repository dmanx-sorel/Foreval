              1. Переписаны и ускорены в разы большинство основных  функций с массивами: max, min, sum, sumq, prod,...

              2. Увеличeна производительность функций вычисления действительных полиномов, заданных массивом - poly(Array, RealArg)   
              
> Убрано:

              2. Функция sumi.
 

> Добавлено:

                  sumqs(VX,x)           =  sum ( (VX[i] - x)^2 ) 
                  prods(VX,x)           =  prod( (VX[i]-x) )

              2. Расширены типы аргументов в процедурах копирования массивов  CopyArray(VDst,VSrc): 
                   Single Array  -> Double Array; Extended Array
                   Integer Array -> Double Array; Extended Array

                                                  v.9.1.1.375

> Добавлено:
   
    1. Ограниченная арифметика указателей: 

        PointerVar                 = LineForm(PointerVars+IntegerVars)
        PointerVar                 = @Array[indx]
        PointerArray[indx]         = LineForm(PointerVars+IntegerVars)
        PointerArray[indx]         = @Array[indx]

    2. Чтение и запись по указателю/адресу:
   
        PType(LineForm(PointerVars+IntegerVars))    
        PType(PointerArray[indx])  
   
        , где
          PointerVar - переменная указатель. 
          PointerArray - массив указателей
          LineForm(PointerVars+IntegerVars) - линейная (расширенная) комбинация указателей и целочисленных переменных
          PType = тип чтения/записи по указателю = {PExtended, PDouble, PInteger, PSingle} 

    3. Копирование массивов и участков памяти соответствующих типов: 

          полное копирование массива ArraySrc в ArrayDst с изменением длины ArrayDst (= ArraySrc ):
             ArrayDst = ArraySrc ; 
          
          полное копирование массива ArraySrc в ArrayDst без изменения длины ArrayDst:
             CopyArray(ArrayDst , ArraySrc);  

          полное копирование массива ArraySrc в ArrayDst (по указателю/адресу) без изменения длины ArrayDst:
             CopyArrayDbl, CopyArrayExt, CopyArrayInt, CopyArraySng (@ArrayDst, @ArraySrc) или (PointerArrayDst, PointerArraySrc) 
          
          частичное копирование (вставка) участка массива сответствующего типа с начальным адресом @AddrSrc по адресу @AddrDst числом ячеек - NumCopyCell 
             CopyMemDbl, CopyMemExt, CopyMemInt, CopyMemSng (@AddrDst,@AddrSrc,NumCopyCell)

    4.  High(Array) {Hi(Array)} - верхний(последний) элемент массива (=Length(Array)-1) 
 
    5. Команды вызова подпрограмм и возврата из них изнутри программы: Gosub(LabelSub); Return();  

    6. Дополнительные внутренние команды вычислений с обработкой и перехватом ошибок :
          SetMaskFPU();                  - маскирует исключения FPU
          ErrorCode=ReadExceptionFPU();  - если исключения есть, то возвращает '1', иначе '0'
          ResetMaskFPU();                - сбрасывает флаги исключений
          ErrorCode=IsNAN(argf);         - возвращает 1, если аргумент argf – не является числом (argf = NAN, ±INF) , иначе '0'

        В цикле вычислений работают быстрей, чем try() - exception()   (См. doc)
   
    7. Дополнительные внешние команды вычислений с перехватом FPU ошибок
          flSetExtAddrErrorFPU(@FPUError);  принимает адрес  переменной для  возврата кода ошибки FPU при вычислении выражения в следующих командах:
               flResultMaskedFPU(@CFunc);                                                     
               flResultMaskedFpuE(@CFunc, @Res);
               flResultMaskedFpuD(@CFunc, @Res);  
               flResultMaskedFpuCxE(@CFunc, @Re,@Im);                              
               flResultMaskedFpuCxD(@CFunc,  @Re, @Im);  
               flResultMaskedFpuCxEP(@CFunc,@Res);  
               flResultMaskedFpuCxDP(@CFunc,@Res);   

       В цикле вычислений работают быстрей, чем flResultSafe команды.  (См. doc)

    8. flIsNAN(val:extended):integer - проверка val на NAN,INF  

    9.  flSetParamI(Name, IParam) - задание целочисленных параметров. 

    10.  flCompileATE(Expr, Attr, var Addr, var RetType, var Error) - компиляция выражения с возвратом: адреса , типа результата, ошибки
         
    11. Выбор формулы для вычисления arccotan:
           стандартный:
              аrccotan(x) = pi/2 - arctan(x);  
           инверсный:           
              arccotan(x) = arctan(1/x) ;
